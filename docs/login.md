# cu login

Авторизация через браузер. Открывает Chrome, перенаправляет на Keycloak SSO. После входа cookie сохраняется автоматически.

## Использование

```bash
cu login
cu login --timeout 10m
```

## Флаги

| Флаг | По умолчанию | Описание |
|------|--------------|----------|
| `--timeout` | `5m` | Таймаут ожидания входа |

## Как это работает

1. Открывается окно Chrome с `my.centraluniversity.ru`
2. Вы входите через Keycloak (логин/пароль или SSO)
3. CLI автоматически перехватывает `bff.cookie`
4. Cookie сохраняется в `~/.cu-cli/cookie`

Все последующие команды используют сохранённый cookie.

Если cookie истёк — просто выполните `cu login` ещё раз.

## Альтернативный способ

Можно установить cookie вручную через переменную окружения:

```bash
export CU_BFF_COOKIE="значение-cookie"
```

Переменная окружения имеет приоритет над сохранённым файлом.
