# cu materials

Скачивает PDF-файлы курса и показывает ссылки на внешние ресурсы (GitLab, Notion).

## Использование

```bash
cu materials <course> [flags]
```

## Флаги

| Флаг | По умолчанию | Описание |
|------|--------------|----------|
| `--links` | `false` | Только показать ссылки и файлы, без скачивания |
| `--week N` | `0` (все) | Скачать только конкретную неделю |
| `--path DIR` | `.` | Папка для сохранения |

## Примеры

### Скачать все материалы

```
$ cu materials алгоритмы --path ./downloads
Materials: Алгоритмы и структуры данных. Часть 2

[Неделя 1: Основы теории графов]
  saved: Неделя 1_Алгоритмы и структуры данных. Часть 2_Лекция.pdf
  saved: Неделя 1_Алгоритмы и структуры данных. Часть 2_Семинар.pdf
  saved: Неделя 1_Алгоритмы и структуры данных. Часть 2_Семинар_Указания.pdf

[Неделя 2: DFS, BFS, 0-k BFS]
  saved: Неделя 2_Алгоритмы и структуры данных. Часть 2_Лекция.pdf
  ...

Downloaded 32/32 files
```

### Только посмотреть, что есть

```
$ cu materials java --links
Materials: Java Spring (...)

[Неделя 1: Экосистема Spring]
  [PDF] Неделя 1_Java Spring_Лекция.pdf (9595.1 KB)
  [link] https://git.culab.ru/acs/java-spring-2026/-/blob/main/week_01/longread_01.md
  [link] https://git.culab.ru/acs/java-spring-2026/-/blob/main/week_01/seminar_01.md
```

### Скачать только одну неделю

```
$ cu materials go --week 8
Materials: Разработка на языке программирования Go

[Неделя 8: Контексты. Часть 2]
  saved: Неделя 7-8_Разработка на языке программирования Go_Лекция.pdf
  [link] https://git.culab.ru/.../week08-context-part2/seminar.md
  [link] https://git.culab.ru/.../week08-context-part2/lecture.md

Downloaded 1/1 files
```

## Структура папок

Файлы сохраняются в структуре:

```
./downloads/
  Алгоритмы и структуры данных. Часть 2/
    02-Перезачет/
      Заявление на перезачет.docx
    03-Неделя 1- Основы теории графов/
      Лекция.pdf
      Семинар.pdf
      Семинар_Указания.pdf
    04-Неделя 2- DFS, BFS, 0-k BFS/
      ...
```

## Типы материалов

| Тип | Описание |
|-----|----------|
| `[PDF]` | PDF-файл лекции, семинара или указаний — скачивается автоматически |
| `[link]` | Внешняя ссылка (GitLab, Notion) — выводится в терминал |
