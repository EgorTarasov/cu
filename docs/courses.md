# cu courses

Показывает список всех ваших актуальных курсов с ID.

## Использование

```bash
cu courses
```

## Пример

```
$ cu courses
Your courses (8)

  1. [947] Java Spring (Разработка веб-приложений на Java с использованием Spring)
  2. [824] Soft Lab. Gogol School
  3. [950] SQL и базы данных (для разработчиков)
  4. [901] Алгоритмы и структуры данных. Часть 2
  5. [780] Практика в формате буткемпа
  6. [948] Промышленная разработка. Магистратура
  7. [733] Пульс-опрос. Магистратура
  8. [900] Разработка на языке программирования Go

Use course ID or name with other commands:
  cu deadlines go
  cu grades алгоритмы
  cu materials 901
```

## Поиск курса по имени

Все команды, принимающие курс как аргумент, поддерживают поиск:

| Ввод | Результат |
|------|-----------|
| `900` | Курс с ID 900 |
| `go` | "Разработка на языке программирования **Go**" |
| `sql` | "**SQL** и базы данных" |
| `алгоритмы` | "**Алгоритмы** и структуры данных" |
| `spring` | "Java **Spring**" |

Поиск регистронезависимый. При неоднозначности (несколько совпадений) CLI покажет варианты и попросит уточнить.
