# cu deadlines

Показывает ближайшие дедлайны по всем курсам или по конкретному.

## Использование

```bash
cu deadlines              # все дедлайны
cu deadlines go           # по курсу Go
cu deadlines 901          # по ID курса
cu deadlines алгоритмы    # по названию
```

## Пример — все дедлайны

```
$ cu deadlines
All upcoming deadlines

   Промышленная разработка. Магистратура
 * IN PROGRESS   1d 3h     22 Apr 20:59  ДЗ 5. Неделя 10.
   SQL и базы данных (для разработчиков)
 * IN PROGRESS   2d 1h     23 Apr 19:00  ДЗ 3. Неделя 8.
   Разработка на языке программирования Go
   reviewer: Владислав Ханнанов
 * TODO          2d 3h     23 Apr 20:59  ДЗ 4. Неделя 10.
   Разработка на языке программирования Go

15 deadline(s) total
  ! = overdue or <24h  * = <3 days
```

## Пример — по курсу

```
$ cu deadlines sql
Deadlines: SQL и базы данных (для разработчиков)

 * IN PROGRESS   1d 3h     22 Apr 20:59  ДЗ 5. Неделя 10.

1 deadline(s) total
```

## Обозначения

| Символ | Значение |
|--------|----------|
| `!` | Просрочено или осталось менее 24 часов |
| `*` | Осталось менее 3 дней |
| (пусто) | Более 3 дней |

## Статусы задач

| Статус | Значение |
|--------|----------|
| `TODO` | Не начато |
| `IN PROGRESS` | В работе |
| `SUBMITTED` | Отправлено на проверку |
| `DONE` | Оценено |
| `FAILED` | Просрочено |
